Definition of programming

Programming is the act and effect of programming. This verb has several uses: it refers to the design and indicates the actions to be taken during the project; prior to the notification of the parts that make up the act or performance; preparing machines to perform a specific task at a given moment; developing programs for solving problems using computers; and preparing the necessary data to solve the problem.

Programing languages

Each program is written in one or another programming language, which is then translated into machine code understood by the computer. While it is possible to program directly in machine code, high-level programming languages ​​greatly simplify the development process. There are many different types and levels of complexity of programming languages, so each language is better suited to specific tasks.

Application development

Application development is a complex process and programming is only a small part of the process. The main and most common parts of development: Collection and analysis of requirements Design and selection of the most appropriate tools (programming languages, platforms, data sheets, etc.) to solve the problem. Writing a program in the selected programming language. Testing. If testing reveals that the desired quality level has not been reached, correct the problem (return to step 1) Documentation, translation into other languages ​​if necessary Support Installation
